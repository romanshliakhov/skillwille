# Teams for development and production

**INSTALL DEPENDENCIES** - Run command: `npm i`

1. **Developer Mode**
   Run command: `npm run dev`

2. **Production mode**
   Run command: `npm run build`

3. **Production mode and creation of a ZIP archive with the result**
   Run command: `npm run zip`

4. **Production mode without creating WEBP images and actions related to this format**
   Run command: `npm run devbuild`
